app.post('/reporte', (req, res) => {
  const mensaje = req.body.mensaje;
  res.json({
    estado: "Reporte recibido",
    mensaje: mensaje
  });
});

¿Qué hace esta ruta?
Esta ruta utiliza el método HTTP POST para recibir información que viaja oculta y estructurada en el cuerpo de la solicitud (req.body). El servidor extrae la propiedad 'mensaje', la procesa y devuelve una respuesta estructurada en formato JSON ({ estado: "Reporte recibido", mensaje: ... }). A diferencia de GET, esta ruta está diseñada para registrar, guardar o alterar el estado del sistema.

¿Para qué podría servir en una aplicación comunitaria?
En un entorno social o comunitario, este endpoint es ideal para canales de participación ciudadana y veeduría. Podría utilizarse para recibir reportes de fallas en servicios públicos, denuncias de vulneración de derechos, alertas de emergencias locales, o simplemente para enviar solicitudes y propuestas de los ciudadanos directamente hacia los administradores de la comunidad, garantizando que la información viaje de forma segura y llegue organizada al backend.
