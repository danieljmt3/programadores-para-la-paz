El comando "uname -a" muestra información detallada sobre el sistema operativo, como el nombre del sistema, la versión del kernel, la arquitectura del procesador y otros datos relacionados con el equipo.

El comando "whoami" muestra el nombre del usuario que actualmente ha iniciado sesión en el sistema.

Conocer esta información es importante cuando se administra un servidor, ya que permite verificar el sistema operativo utilizado, identificar el usuario con el que se están ejecutando los comandos y facilitar tareas de mantenimiento, configuración y solución de problemas.
